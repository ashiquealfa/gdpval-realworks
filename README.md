## 📋 About gdpval-realworks

gdpval-realworks helps you test large language models (LLMs) on real professional tasks instead of simple puzzles. It uses YAML files to set up tests and shows results in a live dashboard built with React. This project focuses on 220 different tasks from 11 industries to give a clear view of model performance in real work environments.

You do not need programming skills to use this software. Follow the steps below to get it running on your Windows computer.

## ⚙️ Configuring Experiments with YAML Pipelines

gdpval-realworks runs tests using YAML files. These files tell the software what tasks to run and how. You don’t need to create your own YAML files unless you want to try custom tests.

The app includes preset YAML files for the GDPVal Gold Subset covering 220 tasks across various industries like finance, healthcare, legal, marketing, and more.

If you want to adjust or create YAML files, follow these tips:

- Open the included YAML files in a simple text editor like Notepad.
- Each file lists tasks and parameters in plain text.
- Do not change file structure unless you understand YAML syntax.

Users can load these files through the dashboard by clicking “Load Experiment” and browsing to the YAML file location.
